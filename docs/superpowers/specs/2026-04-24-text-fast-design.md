## Summary

text-fast.com is a free online text tools hub monetized by Google AdSense. It follows the same model as convert-fast.com and finance-fast.com: many focused tool pages, each targeting a long-tail keyword, built with the same React + Vite stack. Users paste text, get instant results, and leave ‚Äî no sign-up, no server calls.

---

## Goals

- Reach $1,000‚Äì$2,000/month AdSense revenue within 12‚Äì18 months.
- Launch with 10 tools (Sprint 1), expand to 30+ tools over subsequent sprints.
- Apply for AdSense after the site establishes organic traffic, following the same pattern as convert-fast.com.
- Add i18n (top 20 languages) after AdSense approval and 3‚Äì4 months of organic data.

### Data Flow

1. User pastes or types text into `TextInput`.
2. `ToolPage` passes the value to the relevant pure function in `src/tools/`.
3. `ResultPanel` renders the output in real time (on every keystroke via `useMemo`).
4. No API calls, no localStorage, no state management library.

## UI Principles

- **Real-time output.** Results update on every keystroke. No submit button.
- **Copy button** on every output panel (Clipboard API).
- **Dark mode.** CSS custom properties switch between light and dark token sets. A toggle in the header persists the preference to `localStorage`. Default follows `prefers-color-scheme`.
- **Mobile-first.** Single-column layout. Full-width textarea on small screens.
- **AdSense slots.** Placeholder `<div>` elements positioned below the header and below the tool. Activate only after approval.

```css
/* Light */
```

## SEO

Every tool page includes:

- Unique `<title>` and `<meta name="description">`
- `FAQPage` JSON-LD (3‚Äì5 questions per tool)
- `WebApplication` JSON-LD

- `noindex` on the 404 page only

`public/sitemap.xml` generated by `generate-sitemap.mjs` (same script pattern as convert-fast.com). Submitted to Google Search Console after launch.
